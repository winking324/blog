VirtualBox 安装 Ubuntu 18.04

2018-10-19

虚拟机软件有很多种，这里使用 VirtualBox 作为主要工具，来说明如何安装 Ubuntu 18.04。

1 下载 Ubuntu 镜像

  • Ubuntu releases

2 安装

  1. 点击新建，输入名称并配置各个选项后，点击创建；

  2. 选择虚拟机存储位置和存储大小，之后点击创建；

  3. 选择刚创建的虚拟机，进行设置，在存储的光驱控制器上，选择已经下载好的 Ubuntu 镜像文件；

  4. 选择刚创建的虚拟机，点击启动按钮，进入系统安装界面，选择语言后回车；

  5. 选择键盘布局，回车；

  1. 选择安装，回车；

  2. 选择网卡，回车；

  3. 如果有代理，输入代理地址，回车；

  4. 默认，回车；

  5. 使用整个硬盘，回车；

  6. 初始化硬盘，回车；

  7. 默认，回车；

  8. 选择继续，进行硬盘初始化，回车；

  9. 输入用户名和密码等，回车；

  10. 空格选择需要安装的应用，回车继续；

  11. 等待安装结束；

  12. 安装结束，重启系统；

  13. 回车移除安装镜像，并自动重启；

  14. 安装成功，进入登录界面，输入配置的用户名和密码；

  15. 进入 Ubuntu 系统；
